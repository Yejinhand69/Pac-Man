A modern remake of the classic **Pac-Man** game, developed using Unity.  
This version stays true to the original gameplay while adding polished visuals, smooth animations, and refined controls.

## Features
- Classic maze-chase gameplay
- Ghost AI behavior (scatter, chase, frightened)
- Pellet and power-up mechanics
- Scoring system and lives
- Retro-inspired UI

## How to Play
- Use arrow keys or WASD to control Pac-Man.
- Eat all the pellets to complete the level.
- Avoid the ghosts — unless you've powered up!

## Technologies
- Unity (C#)
- 2D Game Development
- Custom AI and pathfinding
